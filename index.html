Jimmy V's Grill & Pub

www.jimmyvspub.com

The Jimmy V's Grill & Pub web application is a freelance project I undertook in early-2017. Built with PHP and the CodeIgniter framework, this application features an MVC architecture with a database-driven backend allowing employees with login credentials to manage the online menu. Employees can also use the admin dashboard to create daily specials on the homepage that are then removed with a scheduled Cron script at the end of each night. Additionally, the site utilizes a custom PHP wrapper for making calls to the Untappd API to display the restaurant's beer list. As the only developer, I was responsible for all aspects of the project including design, deployment, and maintenance.

Brewery District Browns Backers

www.brownsbackerscolumbus.com

The Brewery District Browns Backers application was a freelance project started in early-2017 and developed in conjunction with the Jimmy V's Grill & Pub page. As Jimmy V's is the official sponsor of the Brewery District Browns Backers, the owner (and club president) desired a separate application to handle club events and membership. This application was developed with the CodeIgniter PHP framework and features backend administrative tools for managing the club's mailing list. The admin dashboard features CRUD operations for editing member information, as well as a WYSIWYG editor (TinyMCE) for composing HTML emails. Emails can then be sent to any number of members and are delivered in small batches using a Cron task scheduler to adhere to email limitations imposed by the hosting provider and avoid blacklisting by spam filters. Altogether, this project provided an excellent opportunity to explore the importance and best practices of online marketing.

Cryptocurrency Market Tracker & Alert System

http://ss-trading.hopto.org

The cryptocurrency market tracker and alert system is an on-going personal passion project I began in early-2017 to track and store cryptocurrency market data, events, and conditions. This project initially began with the development of PHP wrapper classes for several major crytpocurrency market APIs that were used in a scheduled Cron script to gather each exchange's recent transaction data to be stored in a time-series database (InfluxDB) set up on a Raspberry Pi. As the project evolved, I began using this data to implement a system for analyzing current market conditions to send email alerts for certain market events such as arbitrage opportunities, volume spikes, and large price swings. Finally, to pull everything together, I began developing a web interface using the Laravel PHP framework to provide data visualizations and tools for the customization of alert triggers. Although this project is still very much a work in progress, it has given me the ability to combine my loves for cryptocurrency and development while experiencing a variety of full-stack development roles including database design/administration, server administration, and interface design.
